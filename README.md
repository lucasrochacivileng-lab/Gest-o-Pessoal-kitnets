# KitManager Pro - Seed para VS Code

Este é um scaffold inicial em React + Vite + Tailwind para recriar fora da Base44 o app KitManager.

## Rodar localmente

```bash
npm install
npm run dev
```

## Próximo passo para IA do VS Code

Use o arquivo `PROMPT_PARA_IA.md` como instrução para o Copilot/Cursor/Codex no VS Code.

## Estrutura já criada

- Layout com sidebar inspirado na Base44
- Rotas principais
- Dashboard inicial
- Módulo genérico para entidades
- Recebimentos com ação simples de receber aluguel
- Cliente local mockado em memória

## Entidades mapeadas

Kitnet, Tenant, Contract, Receivable, Payment, Expense, Construction, CreditCard, Document, PersonalIncome, ExpertReport, ComplementaryProject, User.
